# Project Title
Blümeo
# Project benefit
BLÜMEO – a digital platform that connects private customers and local flower shops.
Private customers can create individually designed bouquets using a 3D editor and conveniently order them from local providers.
Flower shops gain access to digitalization through BLÜMEO by being able to manage their assortment digitally, become visible without having their own website, benefit from better customer retention, explore new advertising opportunities, and further develop themselves in an increasingly competitive market.
Value Hypotheses:
Private customers:
If we offer a platform with a 3D editor, customers will use it to design and order flowers from local shops independently of their location and with just a few clicks.
Flower shops:
If we offer an easy-to-use app for product and order management, flower shops will use it to become visible online and reach new customer groups without technical knowledge or effort.
# Target group
1. Private customers: Individuals who want to order flowers for personal occasions
2. Local Flower Shops (Business customers)
